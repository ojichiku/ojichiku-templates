# WordPress 向けプライバシーポリシー・利用規約テンプレート

このフォルダには、WordPress サイトで利用できる **プライバシーポリシー** と **利用規約** の Markdown テンプレートを収録しています。  
おじちくAIラボの記事「プライバシーポリシー・利用規約テンプレート配布」で紹介した内容を、GitHub上で直接参照・利用できる形にまとめています。  

## 含まれるテンプレート

| 種類                       | ファイル名                                                      | 主な用途                                                                                             |
| -------------------------- | --------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- |
| 個人サイト・ブログ向け     | `personal_privacy_policy.md` / `personal_terms_of_service.md`   | 趣味ブログや個人発信サイトでの利用を想定しています。広告やアクセス解析を使う人向けの内容です。       |
| 法人サイト向け             | `corporate_privacy_policy.md` / `corporate_terms_of_service.md` | 企業や事業向けのサイトを想定しています。個人情報保護、業務委託など法人運営に関わる項目を含みます。   |
| ECサイト・会員サービス向け | `ec_privacy_policy.md` / `ec_terms_of_service.md`               | 商品販売や会員制サービスに対応したテンプレートです。決済、キャンセル、免責事項などを追加しています。 |

各テンプレートは **Markdown形式** で作成しており、WordPress の固定ページにそのまま貼り付けて利用できます。  

## 利用方法

1. このリポジトリのページを開き、利用したいテンプレートファイルをクリックします。  
2. 表示された画面の右上にある **「Copy raw file」** ボタンを押すと、ファイルの内容がコピーできます。  
   または、**「Download raw file」** でファイルをダウンロードします。
3. WordPress の「固定ページ」にファイルの内容を貼り付けて保存します。  
4. サイトの運営内容に合わせて、事業者名、連絡先、サービス名などを適宜書き換えてください。  

例：  
- 「おじちくAIラボ」 → 自分のサイト名に変更  
- 「info@example.com」 → 自分のメールアドレスに変更
