# おじちくAIラボ テンプレート集

このリポジトリは、おじちくAIラボ（Ojichiku AI Lab）が公開しているテンプレート集です。
Python、PHP、Javaなどの開発テンプレートに加え、ブログ執筆やプロフィール作成に役立つ素材も含まれています。
個人の学習、発信、副業活動など、自由に活用することができます。

## テンプレート一覧

| カテゴリ   | パス                                  | 説明                         |
| ------ | ----------------------------------- | -------------------------- |
| Python | [/python/general](./python/general) | WSLとGitHubで使える一般的なプロジェクト構成 |

## 利用について

このテンプレート集は、自由に利用、改変、再配布できます。
ライセンスは MIT License です。
ただし、再配布や記事で紹介する際は、次のいずれかの形で「出典」を添えていただけると嬉しいです。

* 「テンプレート出典：おじちくAIラボ（[https://www.wanchiku.com）」](https://www.wanchiku.com）」)
* GitHubリポジトリへのリンクを記載
  → [https://github.com/ojichiku/ojichiku-templates](https://github.com/username/ojichiku-templates)

## 目的

生成AI、プログラミング、ブログ発信などを楽しみながら続けるための「土台」を提供することを目的としています。
すぐに使える形に整え、記事と連動しながら学べる構成にしています。
